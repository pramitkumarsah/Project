## Project Summary

This project involves building and securing a small business network using **OpenWRT** (as a Linux router/firewall) running inside **VirtualBox**. The work covers:

- Network setup and configuration with a hosted business website
- Firewall rule configuration and demonstration
- Security hardening of the OpenWRT system
- Network traffic capture and analysis using tcpdump and Wireshark
- Cyber security risk assessment using the TVAMatrix methodology
- Recommended security controls for the highest-risk data asset
